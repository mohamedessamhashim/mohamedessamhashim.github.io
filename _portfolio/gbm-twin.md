Built a comprehensive machine learning pipeline to segment brain tumors in MRI scans using the BraTS 2024 dataset.

- Processed over 1,600 MRI cases with a standardized workflow including alignment, resampling, and normalization.
- Trained and validated deep learning models (PyTorch, CUDA) for tumor region identification with 3D segmentations.
- Exported outputs in clinical format (NIfTI) for radiologist interpretation.
- Released a reproducible codebase for extension and benchmarking.
